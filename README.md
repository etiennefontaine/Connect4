# Puissance_4
Projet C++ - Jeu de Puissance 4 - ENSAE 2A

Voici le projet de C++ réalisé par Etienne Fontaine et Claire Rogier.
Nous avons réalisé le projet sous CodeBlocks, les fichiers .cpp des classes sont dans le dossier src
et les fichiers .h dans le dossier include. En lançant le project file Puissance_4, l'inclusion de ces
fichiers est automatique. Le projet a besoin des fichiers grille, icone, pionsjaune, pionrouge, times et des
.dll Sdl et SDL_ttf pour compiler. Le .exe est situé dans bin\Release.

Nous avons inclus les dossiers compressés des librairies SDL et SDL_ttf dans le dossier Librairies SDL et SDL_ttf.
Ces librairies sont conçues pour CodeBlocks, les lien de téléchargement de ces librairies sont :
https://www.libsdl.org/download-1.2.php pour SDL
https://www.libsdl.org/projects/SDL_ttf/release-1.2.html pour SDL_ttf

Un tutoriel pour l'installation de la librairie SDL est disponible ici : http://openclassrooms.com/courses/apprenez-a-programmer-en-c/installation-de-la-sdl
Idem pour SDL_ttf : http://openclassrooms.com/courses/apprenez-a-programmer-en-c/ecrire-du-texte-avec-sdl-ttf
Attention, il semble que la version 32 bits soit à privilégier pour SDL_ttf
